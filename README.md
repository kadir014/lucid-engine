# lucid-engine
Python Game Engine based on Pygame, ECS and flexibility



# Installation
You can install via PyPI:
```shell
$ pip install lucid-engine
```

... and now what?

You can actually create an empty project with the initializer script! Create a new folder, let's say "MyGame", and cd into it. Then:
```shell
$ python -m lucid_init
```

And done! You have an empty Lucid Engine template to start developing your game.
